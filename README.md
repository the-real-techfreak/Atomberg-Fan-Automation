# Tasmota-Atomberg-Daikin

The intent of this project is to create a DIY IR Module that can control the Cieling Fan and AC Unit in a room using Tasmota Firmware
A quick high level information of the project. Please note that these are just examples and this project can be optimized for any Fan+AC setup that supports IR)

# Hardware Information

Appliances to be controlled:
- DAIKIN AC
- Atomberg (Atomberg Efficio Alpha 1200 / Atomberg Renesa 1200)
  
Components Used (I used these as I already have these components for a different project):
- D1 Mini ESP32 (https://robu.in/product/mini-kit-esp32-wifi-and-bluetooth-module/)
- IR Transmitter (https://robu.in/product/38khz-infrared-transmit-sensor-module/)
- IR Receiver (https://robu.in/product/infrared-receiver-module-arduino/)
